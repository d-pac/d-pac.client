# d-pac

> Digitaal platform voor het assessment van competenties

**NOT PRODUCTION READY**

Not by a long shot.

## Installation

```shell
$ npm install && bower install
```

## Building

```shell
$ grunt build
```

## Previewing

```shell
$ grunt serve
```

## Publishing the docs

```shell
$ grunt publish
```

### Deployment

```shell
$ grunt deploy
```
